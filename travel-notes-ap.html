📱 游记分享 API 接口文档

📌 基础信息
- 基础URL: http://你的IP:3000
- 所有接口返回格式: JSON
- 需要登录的接口请在请求体中包含 user_id
- 图片上传大小限制: 最大5MB，支持jpg/png格式

📖 一、游记接口

GET 获取游记列表（支持分页和筛选）
/api/notes

请求参数：

参数名 类型 必填 说明
page number 否 页码，默认1
limit number 否 每页数量，默认10
scenic_id number 否 按景点ID筛选

请求示例：

/api/notes?page=1&limit=10
/api/notes?scenic_id=5
/api/notes?page=2&limit=5&scenic_id=3

成功返回示例：

{
  "notes": [
    {
      "id": 1,
      "title": "九寨沟游记",
      "content": "今天的九寨沟特别美...",
      "cover_img": "http://localhost:3000/uploads/cover1.jpg",
      "scenic_id": 5,
      "user_id": 3,
      "like_count": 25,
      "create_time": "2026-03-09T10:30:00.000Z",
      "username": "张三",
      "scenic_name": "九寨沟"
    }
  ],
  "pagination": {
    "page": 1,
    "limit": 10,
    "total": 50,
    "pages": 5
  }
}
GET 获取游记详情（含评论）
/api/notes/{id}

请求示例：

/api/notes/1

成功返回示例：

{
  "id": 1,
  "title": "九寨沟游记",
  "content": "今天的九寨沟特别美...",
  "cover_img": "http://localhost:3000/uploads/cover1.jpg",
  "scenic_id": 5,
  "user_id": 3,
  "like_count": 25,
  "create_time": "2026-03-09T10:30:00.000Z",
  "update_time": "2026-03-09T10:30:00.000Z",
  "username": "张三",
  "scenic_name": "九寨沟",
  "comments": [
    {
      "id": 1,
      "content": "写得太好了！",
      "create_time": "2026-03-09T11:00:00.000Z",

失败返回：

{
  "error": "游记不存在"
}
POST 发布游记
/api/notes

请求格式： multipart/form-data

请求参数：

参数名 类型 必填 说明
title string 是 游记标题，不超过100字
content string 是 游记内容
scenic_id number 否 关联的景点ID
user_id number 是 发布者用户ID
cover_img file 否 封面图片（jpg/png，≤5MB）

成功返回：

{
  "success": true,
  "message": "发布成功",
  "noteId": 3
}

失败返回：

{
  "error": "标题、内容和用户ID不能为空"
}
PUT 编辑游记
/api/notes/{id}

请求格式： multipart/form-data

请求参数：

参数名 类型 必填 说明
title string 是 游记标题
content string 是 游记内容
scenic_id number 否 关联的景点ID
user_id number 是 用户ID（用于验证权限）
cover_img file 否 新封面图片（不传则保留原图）

成功返回：

{
  "success": true,
  "message": "修改成功"
}

失败返回：

{
  "error": "只能编辑自己的游记"
}
DELETE 删除游记
/api/notes/{id}

请求体（JSON）：

{
  "user_id": 3,
  "is_admin": false   // 是否为管理员，可选，默认false
}

成功返回：

{
  "success": true,
  "message": "删除成功"
}
GET 获取指定用户的游记列表
/api/users/{user_id}/notes

请求示例：

/api/users/3/notes

💬 二、评论接口

POST 发表评论
/api/notes/{id}/comments

请求体（JSON）：

{
  "user_id": 3,
  "content": "写得太好了！我也想去"
}

成功返回：

{
  "success": true,
  "message": "评论成功",
  "comment": {
    "id": 5,
    "note_id": 1,
    "user_id": 3,
    "content": "写得太好了！我也想去",
    "create_time": "2026-03-09T12:00:00.000Z",
    "username": "张三"
  }
}

失败返回：

{
  "error": "评论不能超过500字"
}
GET 获取游记的评论列表
/api/notes/{id}/comments

请求示例：

/api/notes/1/comments

成功返回：

[
  {
    "id": 1,
    "note_id": 1,
    "user_id": 4,
    "content": "写得太好了！",
    "create_time": "2026-03-09T11:00:00.000Z",
    "username": "李四"
  }
]
DELETE 删除评论
/api/comments/{id}

请求体（JSON）：

{
  "user_id": 4,
  "is_admin": false
}

成功返回：

{
  "success": true,
  "message": "删除成功"
}

👑 三、管理员审核接口（需管理员登录）

GET 管理员获取所有游记（带审核状态）
/api/admin/notes?status=pending&page=1&limit=20

功能说明：管理员查看所有游记，可按状态筛选

参数：

参数名 类型 必填 说明
status string 否 pending(待审核)/approved(已通过)/rejected(已拒绝)
page number 否 页码，默认1
limit number 否 每页数量，默认20

请求示例：

/api/admin/notes?status=pending
    /api/admin/notes?status=approved&page=2&limit=10
PUT 审核游记（通过/拒绝）
/api/admin/notes/:id/review

请求体（JSON）：

{
    "status": "approved",  // approved 或 rejected
    "review_reason": "拒绝原因（拒绝时必填）"
    }

成功返回：

{
    "success": true,
    "message": "游记已通过审核"
    }

失败返回：

{
    "error": "拒绝时必须填写原因"
    }
POST 批量审核游记
/api/admin/notes/batch-review

请求体（JSON）：

{
    "note_ids": [1, 2, 3],
    "status": "approved",  // approved 或 rejected
    "review_reason": "批量审核备注（可选）"
    }

成功返回：

{
    "success": true,
    "message": "成功处理 3 条游记",
    "affectedRows": 3
    }
GET 获取待审核游记数量
/api/admin/notes/pending-count

成功返回：

{
    "count": 5
    }

📝 四、状态码说明

状态码 说明
200 请求成功
400 请求参数错误
403 没有权限
404 资源不存在
500 服务器内部错误

🎯 五、前端调用示例

使用 fetch

// 获取游记列表
async function getNotes(page = 1) {
  const res = await fetch(`/api/notes?page=${page}`);
  return await res.json();
}

// 发布游记
async function publishNote(formData) {
  const res = await fetch('/api/notes', {
    method: 'POST',
    body: formData
  });
  return await res.json();
}

// 发表评论
async function postComment(noteId, userId, content) {
  const res = await fetch(`/api/notes/${noteId}/comments`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ user_id: userId, content })
  });
  return await res.json();
}

使用 axios

// 获取游记详情
axios.get('/api/notes/1')
  .then(res => console.log(res.data))
  .catch(err => console.error(err));

// 删除游记
axios.delete('/api/notes/1', {
  data: { user_id: 3 }
}).then(res => console.log(res.data));
