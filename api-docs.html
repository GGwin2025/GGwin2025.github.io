🌍 旅游推荐网站 API 接口文档

📌 基础信息
- 基础URL: http://你的IP:3000（例如：http://192.168.1.105:3000）
- 所有接口返回格式: JSON
- 数据库已更新：15个地方 + 60个项目（景点30个，美食30个）
- 如果遇到跨域问题，已配置CORS，无需担心

🏞️ 前台接口（供前端调用）

GET 获取所有地方
/api/places

功能说明：获取全部15个地方数据，用于首页展示

返回示例：

[
  {
    "id": 1,
    "name": "沙溪古镇",
    "location": "云南省大理白族自治州剑川县",
    "category": "古城古镇"
  },
  ...
]
GET 获取单个地方及其项目
/api/places/{id}

功能说明：获取某个地方的详细信息及其包含的4个项目

示例： /api/places/1

GET 获取所有项目（景点/美食）
/api/items?type=attraction

功能说明：获取所有项目，可按类型筛选

参数：type - attraction(景点) 或 food(美食)

示例： /api/items?type=food（获取所有美食）

GET 获取单个项目详情
/api/items/{id}

功能说明：获取某个景点或美食的详细信息

示例： /api/items/1

❤️ 用户收藏接口

POST 收藏项目
/api/favorites

请求体：

{
  "user_id": 1,
  "item_id": 5
}
DELETE 取消收藏
/api/favorites

请求体：

{
  "user_id": 1,
  "item_id": 5
}
GET 获取用户收藏列表
/api/users/{user_id}/favorites

示例： /api/users/1/favorites

GET 检查是否已收藏
/api/favorites/check?user_id=1&item_id=5

👤 用户登录/注册接口

POST 用户登录
/api/user/login

请求体：

{
  "username": "user1",
  "password": "123456"
}
POST 用户注册
/api/user/register

请求体：

{
  "username": "newuser",
  "password": "123456"
}

⚙️ 后台管理接口（需管理员登录）

POST 管理员登录
/api/admin/login
PUT 修改地方信息
/api/admin/places/{id}
PUT 修改项目信息
/api/admin/items/{id}
GET 获取所有项目（管理员用）
/api/admin/items

📸 图片上传接口

POST 上传单张图片
/api/upload/single

参数：image - 图片文件

DELETE 删除图片
/api/upload/{文件名}

📊 数据统计

GET 获取网站统计数据
/api/stats

返回示例：

{
  "places": 15,
  "items": 60,
  "attractions": 30,
  "foods": 30,
  "users": 2,
  "favorites": 0
}

🔧 测试指南

1. 测试服务器是否运行：

http://你的IP:3000/api/places

2. 测试后台管理页面：

http://你的IP:3000/admin.html

账号：admin / 123456

3. 数据库信息：

  • 15个地方
  • 60个项目（景点30个，美食30个）
  • 2个测试用户（user1/user2，密码123456）
  • 1个管理员（admin，密码123456）
